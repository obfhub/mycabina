Auto-Upload Setup Guide

Use this simple method to automatically upload photos from your laptop to any event.

Step 1: Install Node.js

Download and install from nodejs.org

Step 2: Get Your Upload Script

The upload script is ready to use! Download it or copy the command below to start uploading photos.

Step 3: Run Upload Command

On Windows: Paste the command into PowerShell or Command Prompt
On Mac/Linux: Paste into Terminal

The script will watch your folder and automatically upload new photos as they arrive!

Example Usage

# Navigate to your photos folder, then run: node photo-uploader.js ~/Pictures/MyEvent event-name password-123 https://mycabina.onrender.com # The script will watch for new images and upload them automatically! # Supported formats: JPG, PNG, GIF, WebP, HEIC, AVIF

How to Use MyCabina Admin

Creating an Event

  1. Click the "Events" tab
  2. Fill in event details (name, passwords, date, location)
  3. Click "Create Event"
  4. Share the guest upload link with friends
  5. Share the gallery link for viewing photos

Setting Up Auto-Upload from Your Laptop

  1. Go to the "Auto-Upload" tab
  2. Select your event
  3. Copy the command shown
  4. Paste it into your terminal/command prompt
  5. The script will watch your folder and upload new photos automatically!

Password Management

  • Upload Password: Used by guests uploading photos. Share this with guests who want to contribute.
  • Gallery Password: Used by guests viewing the gallery. Leave empty if you want the gallery open to everyone.
  • Both passwords are automatically generated and can be changed per event.

Supported Photo Formats

Auto-upload supports: JPG, JPEG, PNG, GIF, WebP, HEIC, AVIF

Troubleshooting

Upload script not working?
Make sure Node.js is installed. Run node --version in your terminal.
Can't see uploaded photos?
Check that photos are in the correct folder. The script will show you when photos are uploaded.
Upload is slow?
Consider compressing images first. Smaller files upload faster!
💡 Pro Tip: Keep your laptop running while the upload script is active. The script will automatically upload new photos as they arrive in your folder.
